Wiktor Grzankowski
------------------
Projekt implementuje interfejs wielomianów rzadkich wielu zmiennych. Program służy jako kalkulator wielomianów. Pełna dokumentacja w formacie Doxyfile jest
dostępna w pliku MainPage.dox oraz po wykonaniu kolejno instrukcji
mkdir release
cd release
cmake ..
make
make doc
-----------------
Project implements interface of polynomials of multiple variables. Program is to be used as a calculator of the polynomials. Full documantation in Doxyfile format
is avalible in the MainPage.dox file and after generating the documentation by typing
mkdir release
cd release
cmake ..
make
make doc
